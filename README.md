# 📚 Project CRUD Buku - Laravel

## 📌 Deskripsi
Project ini adalah aplikasi CRUD (Create, Read, Update, Delete) data buku menggunakan Laravel dan MySQL.

## ⚙️ Fitur
- Tambah data buku
- Tampilkan data buku
- Edit data buku
- Hapus data buku

## 🛠️ Teknologi
- Laravel 8
- PHP
- MySQL
- CSS

## 🚀 Cara Instalasi

1. Clone repository
git clone https://github.com/zaskiaazahra/project-crud-buku.git

2. Masuk ke folder project
cd project-crud-buku

3. Install dependency
composer install

4. Copy file env
copy .env.example .env

5. Generate key
php artisan key:generate

6. Setting database di file .env
DB_DATABASE=crudkia
DB_USERNAME=root
DB_PASSWORD=

7. Jalankan migrate
php artisan migrate

8. Jalankan project
php artisan serve
